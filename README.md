# 🦟 Smart Mosquito Eliminating System

## 📌 Overview
This project is an IoT-based system designed to attract and eliminate mosquitoes without using harmful chemicals.

## ⚙️ Technologies Used
- ESP32
- Embedded C
- IoT

## 🚀 Features
- Eco-friendly mosquito control
- Low power consumption
- Automated operation
